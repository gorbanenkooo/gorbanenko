Это консольная игра «Пятнашки», написанная на Python. Для запуска достаточно интерпретатора Python. Управление осуществляется клавишами «w», «a», «s», «d», соответствующими направлениям вверх, влево, вниз и вправо.

Основные функции программы:

1.Игровая логика — реализован основной цикл и правила игры.

Проверка победы — система определяет, собрана ли головоломка в правильном порядке.

Валидация ввода — проверка корректности вводимых игроком команд.

Механика перемещения — функция сдвига костяшек с проверкой допустимости хода в выбранном направлении.

Генерация игрового поля — реализовано перемешивание двумерного массива, чтобы каждую новую игру начинать со случайной раскладки.
